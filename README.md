# CodeAlpha_Educational_Performance_and_Resource_Allocation  – Power BI

## 📌 Overview

This project was developed as **Task 4** of the **CodeAlpha Power BI Internship**. The dashboard analyzes student academic performance and institutional resource utilization to support data-driven decision-making in educational institutions.

## 🎯 Objective

To build an interactive Power BI dashboard that helps educational institutions:

* Monitor student academic performance
* Analyze resource utilization
* Identify performance and resource gaps
* Support effective education management

## 📊 Dashboard Features

* Total Students, Average Marks, Attendance & Satisfaction KPI Cards
* Average Marks by Department
* Grade Distribution
* Attendance Trend Analysis
* Books Issued Analysis
* Lab Usage Analysis
* Placement Status
* Student Satisfaction by Department
* Interactive Slicers (Department, Semester, Gender)

## 🛠️ Tools Used

* Microsoft Power BI
* Microsoft Excel
* DAX (Basic Calculations)
* Data Visualization Techniques
